# AI Contract

Ten katalog zawiera kontrakt treściowy dla modeli `Claude` i `GPT` dotyczący komunikacji wydarzenia `Rejs Fest 2026`.

## Pliki
- [rejs-fest-2026.contract.json](./rejs-fest-2026.contract.json) - źródło prawdy w formie strukturalnej.
- [CLAUDE_REJS_FEST_2026.md](./CLAUDE_REJS_FEST_2026.md) - prompt-kontrakt przygotowany pod Claude.
- [GPT_REJS_FEST_2026.md](./GPT_REJS_FEST_2026.md) - prompt-kontrakt przygotowany pod GPT.

## Po co to jest
- utrzymuje spójny ton i zasady marki,
- zmniejsza ryzyko dryfu językowego między modelami,
- porządkuje zatwierdzone copy i fakty o wydarzeniu,
- przyspiesza iteracje nad kolejnymi sekcjami strony.

## Jak używać
1. Wybierz plik dla modelu, z którego korzystasz.
2. Wklej go jako instrukcję bazową lub brief roboczy.
3. Dopisz konkretne zadanie, np. "napisz 3 warianty sekcji FAQ" albo "skróć hero pod mobile".

Jeśli potrzebujesz, ten kontrakt można później rozwinąć o sekcje sponsorów, FAQ, regulamin, maile transakcyjne albo social copy.
